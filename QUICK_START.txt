⚡ QUICK START (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════

Step 1: Navigate to Project
──────────────────────────
cd iets-lms-mysql

Step 2: Copy Environment File
─────────────────────────────
cp .env.example .env

Step 3: Edit .env with Your MySQL Credentials
──────────────────────────────────────────────
Open .env and update:
  MYSQL_HOST=localhost
  MYSQL_USER=root
  MYSQL_PASSWORD=your_mysql_password
  MYSQL_DATABASE=iets_lms

Step 4: Install Dependencies
────────────────────────────
npm install

Step 5: Initialize Database & Create Schema
────────────────────────────────────────────
npm run init-db

Step 6: Seed Sample Data (1000 Students)
─────────────────────────────────────────
npm run init-data

Step 7: Start Server
────────────────────
npm start

Step 8: Open Browser
────────────────────
http://localhost:5000

Step 9: Login
─────────────
Email: admin@iets.edu
Password: admin123

DONE! ✅ Your LMS is running!

🔧 MYSQL SETUP (If Not Installed)
═══════════════════════════════════════════════════════════════════════════

Windows:
────────
1. Download: https://www.mysql.com/downloads/mysql/
2. Run installer
3. Choose MySQL Server
4. Complete installation
5. MySQL will start automatically

Mac:
────
1. Install Homebrew (if not have): /bin/bash -c "$(curl ...)"
2. brew install mysql
3. brew services start mysql
4. Done!

Linux (Ubuntu/Debian):
─────────────────────
sudo apt-get update
sudo apt-get install mysql-server
sudo mysql_secure_installation
sudo systemctl start mysql
Done!

Linux (CentOS/RHEL):
────────────────────
sudo yum install mysql-server
sudo systemctl start mysqld
Done!

📞 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Problem: MySQL Connection Error
────────────────────────────────
Check:
1. MySQL is running
2. .env credentials are correct
3. Database exists
Try: npm run init-db again

Problem: Port 5000 Already in Use
──────────────────────────────────
Solution:
1. Change PORT in .env
2. Or kill process on port 5000

Problem: npm install Fails
──────────────────────────
Solution:
1. Clear cache: npm cache clean --force
2. Try again: npm install
3. Update npm: npm install -g npm@latest

Problem: Database Error
──────────────────────
Solution:
1. Verify MySQL credentials in .env
2. Run: npm run init-db
3. Run: npm run init-data
